# 02. Core Concept: **각품(刻品)**

### ■ 각품(刻品)이란?

담솔이 정의한 고유 브랜드 언어.
‘새긴다(刻)’ + ‘품(品)’ = **의미가 새겨진 작품**
도장을 “제품(product)”에서 “개인적 예술결과물(piece)”로 재정의하는 단어.

### ■ 각품이 필요한 이유

* 기존 도장 시장 언어(도장/인장/오브제)는 모두 경쟁과 중복
* “오브제”는 스톤피커와의 언어적 충돌이 발생
* 담솔의 본질은 **도장을 만드는 것이 아니라, 의미를 새기는 행위 자체**
* 각품은 담솔이 만든 새로운 카테고리 → 브랜드 세계관 독점 가능

### ■ 각품의 3요소

1. **형(形)**: 돌의 표정과 자연의 결
2. **각(刻)**: 손으로 새기는 조각·깊이·획
3. **품(品)**: 완성된 작품의 품격·상징적 의미

=> “돌의 형 → 사람의 각 → 작품의 품”
=> 기존 도장 업체들이 절대 구현할 수 없는 고유 포지션

# 05. Brand Philosophy

### ■ 철학 01 — 의미의 복원

디지털 시대에 사라진 ‘도장’의 기능 대신
이름·탄생·기억이라는 **본질적 의미**를 되살린다.

### ■ 철학 02 — 유일함의 존중

원석의 결·색·무늬는 모두 고유하다.
담솔은 자연의 유일함을 해석해 작품으로 승화한다.

### ■ 철학 03 — 손의 미학

기계의 속도보다 손의 감각,
정확함보다 **표정**을 우선한다.

### ■ 철학 04 — 시간의 보존

기록은 바래지만 돌은 남는다.
담솔은 **시간을 형태로 보존하는 브랜드**다.
