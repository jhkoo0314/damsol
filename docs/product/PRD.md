# 1. 프로젝트 개요

## 1.1 목적

* 담솔의 세계관(각품, 수제 제작철학, 브랜드 스토리)을 체계적으로 전달하는 **브랜드 사이트 구축**
* 고객이 브랜드 가치를 이해한 뒤, 실제 구매는 네이버 스마트스토어로 자연스럽게 연결하는 **하이브리드 구매 흐름 설계**
* “공예 브랜드”의 감성과 장인의 실제 작업성을 강조함으로써 **신뢰·가치·감성 경험**을 강화하는 것이 목표

---

# 2. 핵심 전제 (Key Principles)

1. **브랜드 스토리가 웹사이트의 중심**

   * 담솔의 세계관(刻品), 철학, 공정, 기술력 등을 깊이 있게 전개

2. **사진 중심의 공예 브랜드**

   * 히어로·공방사진·손·칼·돌·결 같은 실제 촬영된 사진이 브랜드 신뢰를 좌우
   * 초기 개발에서는 *사진 placeholder*로 배치 후, 촬영 후 업데이트

3. **직접 결제는 하지 않는다**

   * 결제는 네이버스토어로 연결
   * 사이트는 “브랜드 경험 + 돌 선택 + 신뢰 제공”에 집중

4. **Next.js 15.5.9 + Tailwind v4 + shadcn/ui로 구축**

   * 공예 브랜드 특유의 ‘절제된 여백·미니멀 디자인’을 구현하는 데 최적화
   * 정적 페이지 기반이므로 속도·운영비 측면에서도 최적

# 3. 사용자 페르소나 (Persona)

### 👩 30대 여성 (주요 고객층)

* 출산 · 아이 기념 · 가족 기념행사에 의미 있는 기록을 남기고 싶음
* “작가의 손”이 만든 공예작품을 선호
* 네이버스토어 결제 경험이 익숙하고 편리성 선호

### 👩‍👦 신생아/가족 고객

* **탯줄도장**의 보존력·기술력·진정성을 중요하게 평가
* 믿을 수 있는 브랜드의 스토리 중심 설명을 선호

### 👤 개인용 이름도장 고객

* 평생 사용할 “유일한 이름표기 제품”에 의미를 부여
* 기계 각인 대비 손각인의 예술적 차이를 이해하고 싶어 함

# 5. 주요 페이지 구성

## 5.1 랜딩페이지 (Home)

### **섹션 구조**

1. **히어로 섹션**

   * 대형 사진 placeholder
   * 문구: “평생 쓸 이름이기에, 하루의 진심을 다합니다.”
   * 버튼: “브랜드 스토리 보기” “돌 선택하기”

2. **각품(刻品) 브랜드 아이덴티티**

   * “도장은 찍는 도구가 아니라, 각(刻)의 예술이다.”
   * “복제할 수 없는 단 하나의 표정”
   * 작업 손·칼 사진 placeholder

3. **담솔의 철학**

   * 자연광 아래 돌 결 클로즈업 사진 자리
   * 철학 핵심 문구 3가지

     * 석결을 읽는다
     * 손으로 새긴다
     * 시간과 존재를 담는다

4. **탯줄도장 기술 섹션**

   * 진공 보존 / 레진 / 무결점 투명도
   * 전용 매크로 사진 placeholder

5. **제작 과정 섹션**

   * (1) 돌 고르기
   * (2) 결 읽기
   * (3) 서체 디자인
   * (4) 각(刻)
   * (5) 연마 · 마감
   * 각 단계별 사진 placeholder

6. **돌 선택 미리보기**

   * Grid 3×3
   * 사진 placeholder (실제 돌 1~9)
   * 버튼: “전체 돌 보러가기”

7. **상품 카테고리 간단 소개**

   * Classic 刻
   * Premium 刻
   * Masterpiece 刻
   * 각 상품 대표 사진 placeholder

8. **네이버스토어 버튼**

   * 🛒 “네이버페이로 구매하기”

# 6. 세부 기능 정의

## 6.1 돌 선택 페이지 (Stone Selection)

### ✔ 기능 요구사항

1. **정적 JSON로 관리 (DB 없음)**

   * 돌 이름
   * 번호
   * 색상
   * 결 특징
   * 상태(판매중/품절)
   * 이미지 URL (placeholder → 실제 촬영 후 업데이트)

2. **그리드 레이아웃 (shadcn/ui + Tailwind)**

   * 3열 or 4열 반응형
   * hover 시 텍스트 오버레이
   * 클릭 시 상세 보기 모달

3. **상세 모달**

   * 돌의 결 클로즈업
   * 구매 가능한 상품(Classic/Premium/Masterpiece) 선택
   * “네이버스토어에서 구매” 버튼

4. **관리 기능 (운영자 측면)**

   * JSON에서 품절 여부만 수동 변경
   * 이미지 교체 시 public 폴더 교체 or URL만 수정

---

## 6.2 상품 영역(Product)

### **카테고리 정의**

* Classic 刻
  → 기본 수제 각인
* Premium 刻
  → 난도 높은 각 + 고급 원석
* Masterpiece 刻
  → 최상급 원석 + 극세각 + 장시간 작업

### **페이지 구성**

* 각 카테고리 소개 문구
* 대표 사진 placeholder
* 추가 사진 placeholder
* 버튼: “네이버스토어 상품페이지 보기”

# 10. 사진 작업 Placeholder 전략

초기 개발에서는 사진이 없는 상태이므로:

* 모든 섹션에 “placeholder gray block” 배치
* Tailwind: `bg-muted h-[400px] rounded-xl`
* 이미지 비율 → *정해진 프레임에 촬영사진 삽입 가능하게 설계*
* 실제 촬영 후 바로 교체만 하면 됨
